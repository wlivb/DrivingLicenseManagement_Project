# نظام إدارة رخص القيادة - Driving License Management System


مشروع متكامل لإدارة عمليات استخراج وتجديد رخص القيادة، تم بناؤه باستخدام لغة **C#** وقواعد بيانات **SQL Server** و لواجهة المستخدم مكتبة **WinForms**، مع التركيز على تطبيق أفضل الممارسات البرمجية.

## 🛠️ الوظائف الرئيسية (Core Functionalities)
إدارة الرخص الدولية والمحلية: دورة حياة كاملة لإصدار، تجديد، وسحب الرخص.

نظام الاختبارات المتعددة: إدارة اختبارات القيادة (نظري، عملي، عيون) مع جدولة المواعيد.

إدارة المستخدمين والصلاحيات: نظام تسجيل دخول آمن مع تحديد صلاحيات لكل مستخدم.

السجل التاريخي للرخص: إمكانية استعراض تاريخ المخالفات والرخص السابقة لأي سائق.


## 🏗️ هيكلية المشروع (Architecture)
المشروع مصمم بنظام **4-Tier Architecture** لضمان سهولة الصيانة والتطوير:
1.  **Presentation Layer (UI)**: واجهات المستخدم باستخدام WinForms.
2.  **Business Logic Layer (BLL)**: الطبقة المسؤولة عن منطق العمل والتحقق من البيانات.
3.  **Data Access Layer (DAL)**: المسؤولة عن الاتصال المباشر بقاعدة البيانات.
4.  **Data Transfer Objects (DTOs)**: لتبادل البيانات بين الطبقات بشكل آمن ومنظم.

## 🚀 المميزات التقنية

مساعد SQL مخصص (Custom SQL Helper): تم بناء كلاس clsSqlHelper باستخدام الـ Delegates والـ Action<SqlCommand>. هذا التصميم المتقدم يقلل من تكرار الأكواد (Boilerplate Code) ويضمن إدارة احترافية للاتصال بقاعدة البيانات.

نمط وسيط نقل البيانات (DTO Pattern): تم تطبيق نمط Data Transfer Objects لفصل البيانات تماماً عن المنطق البرمجي، مما يسهل عملية الصيانة ويجعل النظام قابلاً للتوسع (Scalable).

إدارة قاعدة البيانات (Database Management): تم توفير سكريبت SQL شامل ومنظم لإنشاء الجداول والعلاقات بضغطة زر واحدة، مما يسهل إعداد بيئة العمل لأي مبرمج آخر.

## 🛠️ المتطلبات وكيفية التشغيل
1.  **قاعدة البيانات**: قم بتشغيل ملف `DatabaseScript.sql` المرفق في المستودع على **SQL Server Management Studio** لإنشاء قاعدة `DVLD`.
2.  **الاتصال**: تأكد من تعديل الـ `ConnectionString` في ملف `clsDataAccessSettings.cs` ليتناسب مع اسم السيرفر لديك.
